# ITI REACT Lab 2

## Project: Advanced Todo App

[Live Demo](https://oniaz.github.io/ITI_REACT_lab2/)

### Requirements

- Add task
- Delete task
- Toggle complete
- Filter (all / completed)
- Persist in localStorage
- useEffect for saving

- Multiple pages
    - Home
    - Tasks
    - Profile
- Navigation bar
- Dynamic routing:
    - /task/:id
